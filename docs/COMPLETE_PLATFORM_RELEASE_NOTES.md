# Complete Platform Release Notes

ARTYVERSE X now includes the planned frontend experience layers for public commerce, authentication, collector account, seller operations and marketplace administration. All new role workspaces share the same responsive, motion-aware visual system and support desktop, tablet, mobile and reduced-motion modes.

Backend integrations remain intentionally outside this frontend release: identity provider, payment gateway, order persistence, inventory service, shipping provider, KYC, moderation service and analytics pipeline.
