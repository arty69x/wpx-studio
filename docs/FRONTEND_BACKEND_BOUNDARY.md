# Frontend / Backend Production Boundary

The repository now provides production-polished frontend flows and deterministic experience data. The following capabilities require real backend implementation before a public commerce launch: authenticated sessions, MFA verification, payment authorization, order storage, stock reservation, seller KYC, shipping labels, returns, refunds, commission settlement, audit storage, moderation queues, file uploads, notification delivery and analytics ingestion.

No client-only mock action should be represented as a completed financial, identity or fulfillment transaction in production.
